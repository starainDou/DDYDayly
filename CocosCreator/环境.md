> #### homebrew


> #### node

```
brew link node
brew uninstall node
brew install node

验证按照成功
npm -v
node -v
```

> #### typescript

> #### VSCode

安装VSCode后打开CocosCreator IDE，
只安装一次，点击菜单栏->开发者->VSCode工作流->安装扩展插件。
每个项目执行一次，点击菜单栏->开发者->VSCode工作流->更新智能提示数据。
